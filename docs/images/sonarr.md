## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      sonarr:
        image: @REGISTRY@/sonarr:latest
        container_name: sonarr
        environment:
          - PUID=@PUID@
          - PGID=@PGID@
          - TZ=@TZ@
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@SONARR_CONFIG_PATH@:/config"
          - "/path/to/@TV_PATH@:/tv" # optional
          - "/path/to/@DOWNLOADS_PATH@:/downloads" # optional
        ports:
          - @SONARR_PORT@:8989
        annotations:
          org.freebsd.jail.allow.mlock: "true"
        restart: unless-stopped
    ```


=== ":material-console: Podman CLI"

    ```bash
    podman run -d --name sonarr \
      -p @SONARR_PORT@:8989 \
      --annotation 'org.freebsd.jail.allow.mlock=true' \
      -e PUID=@PUID@ \
      -e PGID=@PGID@ \
      -e TZ=@TZ@ \
      -v @CONTAINER_CONFIG_ROOT@/@SONARR_CONFIG_PATH@:/config \
      -v /path/to/@TV_PATH@:/tv # optional \
      -v /path/to/@DOWNLOADS_PATH@:/downloads # optional \
      @REGISTRY@/sonarr:latest
    ```
