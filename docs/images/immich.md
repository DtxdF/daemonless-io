## Deployment


=== ":material-docker: Podman Compose"

    ```yaml
    services:
      immich:
        image: @REGISTRY@/immich:latest
        container_name: immich
        environment:
          - DB_HOSTNAME=localhost
          - REDIS_HOSTNAME=localhost
          - IMMICH_MACHINE_LEARNING_URL=http://localhost:3003
        volumes:
          - "@CONTAINER_CONFIG_ROOT@/@IMMICH_DATA_PATH@:/data"
          - "@CONTAINER_CONFIG_ROOT@/@IMMICH_ETC_LOCALTIME_PATH@:/etc/localtime"
        restart: unless-stopped
    ```


=== ":material-console: Podman CLI"

    ```bash
    podman run -d --name immich \
      -e DB_HOSTNAME=localhost \
      -e REDIS_HOSTNAME=localhost \
      -e IMMICH_MACHINE_LEARNING_URL=http://localhost:3003 \
      -v @CONTAINER_CONFIG_ROOT@/@IMMICH_DATA_PATH@:/data \
      -v @CONTAINER_CONFIG_ROOT@/@IMMICH_ETC_LOCALTIME_PATH@:/etc/localtime \
      @REGISTRY@/immich:latest
    ```
